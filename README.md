## Project Architecture

 - [NodeJS](https://nodejs.org/en)
 - [AstroJS](https://astro.build/)
 - [TailwindCSS](https://tailwindcss.com/) and CSS
 - [Contenful](https://www.contentful.com/) for content management
 - [Vercel](https://vercel.com/) for deployment

## Running the Code

 1. Install Node version 22 or later.
 2. Run `npm install`.
 3. Create a `.env` file in the root of the project. Ask @mattwells19 for the secret values.
 4. Run `npm start`.
 5. Open `http://localhost:4321/` in your favorite browser.
